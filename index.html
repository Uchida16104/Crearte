Crearte

A creative musical score generation tool using Processing & MIDI

What is Crearte?

Crearte is software that helps to create musical scores using music generated from images and poems. The etymology of Crearte comes from the combination of \"create\" and \"art\". The directory contains sample .mid files, .wav files, .png files, and .txt files. Features are the following:

  • It can import images, poems, and dances into music.
  • It uses simple shapes and math to create graphic scores.
  • You can also move your body to make sound in real time.

Getting Started

Install it via npm or homebrew and start creating generative visuals:

npm install crearte

or

brew install uchida16104/crearte/crearte
